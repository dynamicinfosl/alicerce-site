Título principal entra com `gsap.from()` com split de palavras (efeito stagger de baixo para cima, `y: 80, opacity: 0, duration: 1.2`) - Subtítulo aparece com fade após o título - CTA buttons entram com delay e leve bounce (`ease: "back.out(1.7)"`) - Background com partículas flutuantes ou efeito parallax sutil 2. **Navbar** - Ao fazer scroll para baixo: `gsap.to(navbar, { background: "rgba(26,58,92,0.97)", backdropFilter: "blur(12px)", boxShadow: "...", duration: 0.4 })` - Links com hover underline animado via CSS + GSAP 3. **Scroll Reveal — Seções** - Cada seção usa `ScrollTrigger` com `trigger`, `start: "top 80%"`, `toggleActions: "play none none reverse"` - Cards de serviços entram em stagger: `gsap.from(cards, { y: 60, opacity: 0, stagger: 0.15, duration: 0.8 })` - Números de estatísticas (contadores animados): `gsap.to(counter, { innerText: valorFinal, duration: 2, snap: { innerText: 1 } })` 4. **Cards de Serviços** - Hover com GSAP: `scale: 1.03`, leve `y: -8`, sombra aumenta suavemente - Ícone do card rotaciona levemente ou pulsa no hover 5. **Seção de Orçamento** - Formulário entra com `gsap.from()` com efeito de "surgir" da esquerda - Labels flutuantes (float labels) com transição suave - Botão de submit com efeito de loading animado ao clicar 6. **Parallax** - Imagem ou gradiente de fundo no hero com efeito parallax via `ScrollTrigger` (move em velocidade diferente do conteúdo) 7. **Cursor personalizado** *(opcional mas valorizado)* - Cursor circular que segue o mouse com GSAP `mousemove` + `lag` suave 8. **Loader inicial** - Tela de loading com logo animada que desaparece antes da página carregar (GSAP timeline: logo entra → pausa → tela sobe revelando o site) --- ## ESTRUTURA DO SITE — SEÇÕES OBRIGATÓRIAS ### 1.
